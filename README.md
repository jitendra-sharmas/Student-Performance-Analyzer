# Student Performance Analyzer

## Project Overview
This project analyzes student performance data using Python and Pandas.

## Tasks Performed
- Data Loading
- Data Cleaning
- Average Marks Calculation
- Top 10 Performers
- Subject-wise Highest Scores
- Grade Assignment
- Summary Statistics

## Technologies Used
- Python
- Pandas
- NumPy
- Google Colab

## Dataset Columns
- Maths
- Physics
- Chemistry
- Result
